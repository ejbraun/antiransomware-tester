This list contains the explanation of the flags and the options the default payload can take (case-sensitive)

  1. encr = Encryption - ASY/SYM
  2. trav = Traversal - BFS/DFS/SUB (SUB - Spawns a sub-process for each sub-folder)
  3. writing = Writing the encrypted data to a new file (same name) or temp file - Yes/No/Tmp
  4. ext = Changing the extension of the file - Yes/No
  5. merg = Merge all the files in a folder into a single file - Yes/No
  6. mid = Encrypt just the middle part of the file - Yes/No
  7. sleep = sleep between encryption - Yes/No
  8. small = encrypt small files - Yes/No
  9. large = encrypt large files - Yes/No
  10. default = default 'Yes' is to encrypt all files and default 'No' is to only encrypt 'pdf,doc,jpg' file extensions - Yes/No
  11. root = root folder location (multiple root folder) can be written by delimiting by ';'
